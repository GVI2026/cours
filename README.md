## Consulter le cours en local

Si vous souhaitez parcourir le support hors ligne après avoir cloné ce dépôt, évitez d'ouvrir les fichiers directement en `file://` (les navigateurs bloquent le chargement de certaines ressources dans ce mode).

Lancez plutôt un serveur HTTP local :

```bash
python3 -m http.server 8000 --directory docs
```

Puis ouvrez [http://localhost:8000/](http://localhost:8000/) dans votre navigateur.
